## 👩‍💻 About the Project

This project was completed as part of the **Tata GenAI Powered Data Analytics Job Simulation**. The project focuses on solving a real-world financial problem using data analytics, predictive modeling, business insights, and AI-powered decision strategies.

The objective was to analyze customer financial behavior, identify delinquency risks, and create an intelligent AI-driven collections strategy for improved business outcomes.

---

# 🎯 Problem Statement

Financial institutions often face challenges in identifying customers who may become delinquent before they miss payments.

The goal of this project was to:

✅ Analyze customer financial data  
✅ Identify important risk indicators  
✅ Build predictive logic for delinquency detection  
✅ Create business recommendations  
✅ Design an AI-powered collections strategy  
✅ Include ethical and responsible AI practices

---

# 📊 Task 1: Exploratory Data Analysis (EDA) & Data Quality Assessment

In this stage, the dataset was explored to understand its quality and discover meaningful patterns.

### 🔍 Activities Performed:

- Examined dataset structure and variables
- Checked missing values
- Identified data anomalies
- Analyzed class imbalance
- Discovered customer risk patterns

### 📌 Key Findings:

📍 Dataset contains:
- 500 customer records
- 19 variables

📍 Missing values found:

- Income → 39 missing values
- Loan Balance → 29 missing values
- Credit Score → 2 missing values

📍 Data anomalies:

- Credit Utilization values exceeded expected limits (>100%)

📍 Class distribution:

- Non-delinquent customers → 420
- Delinquent customers → 80

### 💡 Major Risk Indicators Identified:

⚠️ High Credit Utilization  
⚠️ Multiple Missed Payments  
⚠️ Low Credit Score  
⚠️ High Debt-to-Income Ratio  
⚠️ Short Account Tenure

# 🧹 Task 2: Missing Data Handling & Data Preparation

Data preprocessing was performed to improve dataset quality before predictive analysis.

### 🔧 Data Cleaning Techniques:

| Missing Variable | Method Used | Reason |
|-----------------|-------------|---------|
| Income | Median Imputation | Reduces effect of outliers |
| Loan Balance | Median Imputation | Preserves distribution |
| Credit Score | Mean/Median Imputation | Very few missing values |

### ✅ Additional Steps:

- Corrected invalid values
- Handled anomalies
- Prepared data for prediction models
- Improved overall data quality
